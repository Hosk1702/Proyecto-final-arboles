# Mini-Suite de Sistema de Archivos

## Objetivo
Implementar una jerarquía de directorios y archivos usando Árboles Generales.

## Estructura de Datos
- **Nodo:** Objeto con ID, nombre, tipo, contenido y lista de hijos.
- **Persistencia:** Archivo JSON local.

## Funcionalidades (MVP)
1. Crear carpetas y archivos (mkdir, touch).
2. Mover nodos entre carpetas (mv).
3. Renombrar y eliminar nodos (rm).
4. Búsqueda con autocompletado usando Trie.
5. Exportar recorrido en Preorden.
